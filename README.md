# 📖 Project Overview

Retail companies generate large volumes of transactional and inventory data every day. This project demonstrates how raw retail data can be transformed into actionable business insights using an end-to-end analytics workflow.

The project focuses on analyzing:

- Sales Performance
- Inventory Management
- Customer Behavior
- Regional Performance
- Product Performance

The final deliverable consists of interactive Tableau dashboards supported by SQL analysis, Python data cleaning, and Excel reporting.

---

# 🎯 Project Objectives

- Analyze retail sales performance.
- Measure profitability across products and regions.
- Monitor inventory health and turnover.
- Identify products requiring replenishment.
- Understand customer purchasing behavior.
- Compare store performance.
- Build interactive dashboards for business decision-making.
